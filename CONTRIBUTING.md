# Contributing

This repository accompanies a published study; its primary purpose is reproducibility.

- **Issues.** Bug reports, data-source corrections, and reproducibility questions are welcome via GitHub issues.
- **Pull requests.** Keep changes minimal and focused. Each script is standalone and seed-fixed; please preserve reproducibility (do not change random seeds or silently alter data without noting the source).
- **Data.** Every numeric value must trace to a public primary source. New variables require a cited source and a year.
- **Tests.** Run `python3 test_smoke.py` before opening a PR; it must pass.

By contributing you agree your contributions are licensed under the repository's MIT Licence.
